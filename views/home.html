Until this week we got familiar with Node.js and its popular framework Express.js. we also reviewed JavaScript Concept like working with promise object and error handling. we know how to build a simple web server using Node.js with Express.js and how to send Data back from a “get” request. we know what application object, request and response object are. we got familiar with routing that is a core features of express, and also Middleware and its different types.

What is HTML 5?

This site utilizes HTML5: a markup language used for structuring and presenting content on the World Wide Web. HTML5 is the latest version of Hypertext Markup Language, the code that describes web pages. It's actually three kinds of code: HTML, which provides the structure; Cascading Style Sheets (CSS), which take care of presentation; and JavaScript, which makes things happen.
